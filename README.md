# 🎬 MotionCraft AI — Video Generator

> AI Video Generator powered by Magnific API (Kling model)

Convert images to cinematic videos using AI. Built with vanilla HTML + Vercel serverless functions to solve CORS issues.

## 🚀 Deploy ke Vercel (Gratis)

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

1. Fork/import repo ini ke Vercel
2. Deploy (otomatis)
3. Masukkan Magnific API key di web app
4. Generate video! ✨

## Features

- 🎞️ Image → Video (animasi gambar)
- ✍️ Text → Video (teks ke video via 2 langkah)
- ⬆️ AI Image Upscaler (hingga 8x)
- 📱 Responsive design
- 🗂️ History lokal

## API Key

Dapatkan di [magnific.com/api](https://www.magnific.com/api) — **5 EUR gratis** untuk akun baru!
